# Recent Stories Content Source Block

A content source block that returns stories published within a specified number of days. By default, it returns stories from the last 7 days, but this can be configured through the PageBuilder.

## Configuration Options

- **Number of Days**: How many days back to fetch stories from (default: 7)
- **Feed Size**: Number of stories to return (default: 10)
- **Feed Offset**: Pagination offset for fetching more stories
