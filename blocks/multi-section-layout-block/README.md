# Multi Section Layout Block

A layout block that provides a structured page layout with multiple sections:

1. Header (Full Width)
2. Three Column Section
3. Full Width Section
4. Left Rail + Main Content Section
5. Footer (Full Width)

## Features

- Each section has a distinct colored border for visual separation
- Full width sections (header, full-width, footer) extend edge-to-edge with padding
- Three column and content sections have margins on the sides
- Responsive grid layout for the three column and content sections
- Configurable content for each section through PageBuilder

## Usage

Add the Multi Section Layout block to your page and configure the content for each section:

- Header Content
- Three Column Content (array of 3 elements)
- Full Width Content
- Left Rail Content
- Main Content
- Footer Content

Each section can contain any valid block or content.
